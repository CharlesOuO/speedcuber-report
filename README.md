# Charles Lin - Speedcuber Portfolio

> **A personal portfolio and cubing resource website showcasing official WCA records, personal bests, and solving methodologies.**

This repository contains the source code for a personal speedcubing portfolio website. It serves as a central hub to display competition achievements, specifically focusing on the 3x3x3 One-Handed event, while also providing educational resources for cubers of different skill levels.

---

## Features

### 1. Speedcubing Statistics
* **Official WCA Records:** Displays official single and average times across various WCA events (3x3, OH, 2x2, 4x4, 5x5, Clock, Pyraminx, Skewb, Square-1).
* **Practice Personal Bests (PBs):** Tracks unofficial home practice records including Single, Ao5, and Ao12 times.
* **Competition Highlights:** Highlights notable placements and results from official WCA competitions.

### 2. Cubing Guides & Resources
The website includes dedicated pages for learning and improving cubing skills:
* **Notation:** Explanations of standard Rubik's Cube move notations.
* **LBL Method:** A beginner-friendly guide to the Layer-by-Layer method.
* **CFOP Method:** Advanced algorithms and concepts for the CFOP speedsolving method.
* **3x3 OH:** Specific tips and techniques for 3x3x3 One-Handed solving.
