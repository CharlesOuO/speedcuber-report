3x3 One-Handed

My Main Event: Specialized Finger Tricks & RU-Gen Algs.

One-Handed (OH) solving is my specialty. I focus on smooth turning and high TPS. Unlike many solvers, I do not use the table for stability; I prefer a fully floating style with precise grip control.

1. My Finger Tricks

Ring Finger (R / R')

I primarily use my Ring Finger for R' moves (pulling down) and stabilizing the cube.

Index Push (U / U')

For U moves, I use my index finger to push or pull. E.g., Pushing FL to BL for U moves.

L / L'

Perform an x rotation first, then execute as standard U / U' moves.

D / D'

Flick using the Ring Finger.

R2

Pinky + Ring double flick (continuous).

S / S'

S: Ring Finger Hook.
S': Index Finger Flick.

Standard RU-Gen focused algorithms for OH.

Advanced Strategy (COLL): Using COLL (Corners of Last Layer) algorithms can significantly improve OH performance. By solving OLL and permuting corners simultaneously, you are guaranteed to get an EPLL (U, H, Z perms), which are much faster one-handed.
